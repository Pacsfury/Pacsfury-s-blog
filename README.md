# Pacsfury's blog
---

## The idea
This is a special repo for me. 

It's my place to write, explain and think. Here, I want to put my ideas, results, demosntrations, interactive content, etc.

Just a sandbox for me.

## The engine
This is vanilla. No frameworks, just JS, HTML and CSS. In `blogs.js` all posts are saved. 

The file `menu.js` searches all blogs, displays and registers what you click to show it.

More info in the `Hello, World!` post.
